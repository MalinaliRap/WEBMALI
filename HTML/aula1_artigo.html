HTML

O HTML - Hypertext Markup Language é uma linguagem de marcação utilizada na web para definir conteúdo e estrutura de uma página.

O texto em HTML pode ser marcado de várias formas como em negrito pela tag strong ou pela tag b

Podemos dizer que o HTML é a 1a linguagem de marcação usada para web. Mesmo o XHTML ou o XML vem do HTML.
